## Maintenance
- Keep filenames stable so existing Markdown links remain valid.
- Regenerate thumbnails whenever a project UI or branding changes so screenshots stay representative.
- Do **not** add README files to build output folders (`dist/`, etc.); this directory is safe because it contains only static assets tracked in git.
