Null Cipher

Null cipher example

This is just a demo. This cipher is NOT cryptographically secure!

Encryption: Each digit of the numerical key determines how many random characters to add between each letter of the input.

Example: Input "HELLO" and key "12345" might produce "HGEKGLXMMLCVETO".

Enter plaintext to encrypt, or ciphertext to decrypt.
Each digit controls how many random characters are inserted.
Result of encryption or decryption appears here.
