# Sample media

This directory contains a handful of public-domain photos sourced from [publicdomainpictures.net](https://publicdomainpictures.net). They ship with the repository so the slideshow can be demonstrated without hunting for local images.

Feel free to replace the files with your own media when testing; the server discovers images/videos recursively.
