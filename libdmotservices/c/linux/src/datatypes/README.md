# libdmotservices (C/Linux Datatypes)

Core datatype helpers used across the C library.

## Files
- `datatypes.c` — small numeric helpers (e.g., `size_t` to `int` conversion).
- `ringbuf.c` — generic ring buffer implementation with typed wrappers.
