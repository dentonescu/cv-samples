# libdmotservices (C/Linux Header Set)

This folder contains the public `dmot/` headers. Each header is standalone and keeps dependencies minimal.

## Headers
- [`constants.h`](constants.h) — shared constants and defaults.
- [`datatypes.h`](datatypes.h) — safe numeric helpers.
- [`log.h`](log.h) — lightweight logging API.
- [`math.h`](math.h) — random helpers.
- [`ringbuf.h`](ringbuf.h) — generic ring buffer with typed wrappers and optional C11 `_Generic` macros.
- [`signals.h`](signals.h) — signal generation helpers.
- [`string.h`](string.h) — string utility helpers.
- [`time.h`](time.h) — time utilities.
- [`ui.h`](ui.h) — terminal equalizer and ANSI helpers.
- [`version.h`](version.h) — semantic version helpers.
