# libdmotservices

**libdmotservices** is a personal code library containing small, reusable utility classes and functions that I’ve built to make everyday programming tasks easier.
Each module includes unit tests to ensure that the implemented logic behaves as expected.

Although the project began with Java utilities, my long-term goal is to populate it with reusable code from **every language I work with**.
Each language resides in its own subdirectory.
